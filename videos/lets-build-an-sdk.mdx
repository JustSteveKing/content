# Let's Build an SDK

Welcome to this YouTube Livestream, where we will discuss building a PHP SDK for Open Library. In this stream, we will walk through the steps of building an SDK using the `sdk-tools` package on GitHub. Our goal is to create a Library SDK that will allow us to retrieve information on books, authors, and subjects from Open Library. Whether you're a seasoned developer or starting out, this stream will provide valuable insights into building an SDK using PHP. So sit back, relax, and let's get started!
