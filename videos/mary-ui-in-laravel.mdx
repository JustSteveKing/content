# Mary UI in Laravel

Join me for this livestream as we explore Mary UI, and how we can add it to our Laravel application. 

Mary UI is a set of Blade UI Components made for Livewire 3 and styled around daisyUI + TailwindCSS.

We are going to replace our current UI in our multi-tenant application with components from Mary-UI to make maintaining our code base even easier.
