# Tranquil Stay Booking Platform - Handling Payments

Join this livestream as we start a project requested by a subscriber!

The Tranquil Stay Booking System is a specialised web-based platform designed to streamline the reservation process for a single hotel. Utilising the robust and scalable Laravel framework, this project aims to provide an intuitive and efficient booking experience for both hotel guests and administrators.

The platform will cater exclusively to Tranquil Stay Hotel, enhancing its operational efficiency, customer service, and overall booking management.

In this stream we will be starting to look at how we can accept payments, and if we have time - look into the check-in and check-out process, which will use the state machine we built last time.
